MyInsuApp

Descripción del Proyecto

MyInsuApp es una app de escritorio diseñada para mejorar el seguimiento médico y personal de personas con diabetes. 
La aplicación permite un seguimiento exhaustivo de la administración de insulina.

Proposito y Problematica

En esta primera versión vamos a llevar un registro preciso de las zonas en las que el usuario se aplica las inyecciones
de insulina. El proposito es resolver la falta de registro preciso en la rotación de las zonas.

- Problema: El uso repetido de la misma zona para la inyección de insulina puede derivar en complicaciones como la
lipodistrofia, que afecta tanto a nivel estético como a la absorción de la medicación.
- Solución: MyInsuApp permite llevar un registro preciso de donde se producen las inyecciones, con la intención
de favorecer la correcta rotación. Vamos a registrar cada dosis, hora y lugar de punción.


Tecnologías Utilizadas

- Programación: Java para la parte lógica y JavaFX el desarrollo de la interfaz bajo el patrón MVC.
- Bases de datos: Persistencia de los registros.
- Leguaje de marcas: Generación de informes semanales exportables para facilitar el seguimiento y poder
mostrar a personal sanitario. XML con validación XSD.
- Entornos de desarrollo: Control de versiones con Git - GitHub.
