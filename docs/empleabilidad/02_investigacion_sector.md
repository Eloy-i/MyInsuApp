# Investigación del Sector

## 1. Empresas del sector

He investigado estas empresas que operan áreas que entroncan directamente con mis intereses: la gestión de la salud y el
desarrollo de software funcional.

### SocialDiabetes
    https://www.linkedin.com/company/socialdiabetes/
* **Qué hace:** Se dedican al desarrollo de soluciones móviles centradas en la autogestión de la Diabetes Mellitus tipo 1. Sus herramientas permiten mejorar la autonomía del paciente y la gestión de riesgos mediante el análisis de datos.
* **Perfiles que contratan:** Especialistas en desarrollo móvil (iOS/Android), desarrolladores backend y expertos en gestión de datos.
* **Tecnologías:** Inteligencia Artificial (utilizada para modelos predictivos de glucemia), SQL para la persistencia de datos, Swift y Kotlin para el desarrollo nativo.

### NTT Data Spain
    https://www.linkedin.com/company/ntt-data-europe-latam/
* **Qué hace:** Es una multinacional, con sede en Tokio, especializada en servicios de consultoría tecnológica. Es la mayor tecnológica que tenemos en la Región de Murcia y acaban de ganar el contrato para modernizar el sistema informático del Servicio Murciano de Salud (SMS), lo que los convierte en un destino interesante desde el desarrollo de software sanitario.
* **Perfiles que contratan:** Desarrolladores Backend y especialistas en gestión de grandes bases de datos.
* **Tecnologías:** Java como lenguaje principal de backend y SQL para la gestión de bases de datos.

### Minderest
    https://www.linkedin.com/company/minderest/
* **Qué hace:** Es el mejor ejemplo de cómo el talento local puede competir a nivel global. Nacida en Murcia y habiendo crecido desde abajo con recursos propios, hoy es líder europeo en inteligencia de precios. Han desarrollado algoritmos de web scraping que rastrean millones de productos diariamente, permitiendo a las empresas conocer su posición en el mercado en tiempo real.
* **Perfiles que contratan:** Desarrolladores Backend, especialistas en Big Data y desarrolladores de algoritmos de comparación.
* **Tecnologías:** SQL (para el manejo masivo de datos, área que despierta mi principal interés), Python, Java, JavaScript y Machine Learning para el procesamiento inteligente de la información rastreada.
